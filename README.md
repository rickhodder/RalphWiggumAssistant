# RalphWiggumAssistant
A guided chatbot (python) that helps a user define requirements, PRD, etc for a project. Also includes a Ralph Wiggum loop (python) that can implement the requirements.

A guided chatbot that helps a user:
- define a PRD
- refine criteria
- plan a Ralph loop
- kick it off
- monitor the iterations
  
This project strengthens:
- understanding the Ralph loop
- multi‑agent orchestration
- prompt engineering
- planning and execution loops
- persona‑driven UX
- Python integration
